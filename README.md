# Selenium Automation — Login and Search Module

## Project Overview
Automated test scripts for login and search 
functionality using Selenium WebDriver 
and TestNG in Java.

## Website Tested
https://www.saucedemo.com

## Modules Automated
- Login (valid and invalid scenarios)
- Product search and filter
- Shopping cart functionality

## Test Cases
- Total automated test cases: 25+
- Testing types: Functional, Regression

## Tools and Technologies
- Automation Tool: Selenium WebDriver
- Test Framework: TestNG
- Language: Java
- IDE: Eclipse or IntelliJ
- Build Tool: Maven
- Browser: Chrome
- Version Control: Git and GitHub

## How to Run
- Clone this repository
- Open in Eclipse or IntelliJ
- Run TestNG test suite
